# Academic Portfolio

This folder contains a simple academic portfolio for Ajay Vikram P built from the research-facing content in `../portfolio/`.

## Files

- `index.html`: the portfolio page
- `styles.css`: the styling for the page
- `assets/images/ajay-vikram.jpg`: headshot copied from the existing portfolio
- `assets/docs/Ajay_Vikram_Resume.pdf`: resume copied from the existing portfolio

## View locally

Open `index.html` directly in a browser, or run a simple local server from this folder:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

No git repository was initialized in this directory.
